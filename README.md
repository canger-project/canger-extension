## 介绍
这里是苍耳的浏览器插件工具。在你阅读英语文章时，它可以帮助你快速的翻译单词和句子，在你回复帖子时，它可以帮助你翻译为英文并不断优化你的表达。

![demo](demo/demo.gif)

## 特点

- [ ] 划词翻译
- [x] 段落翻译
- [x] 文本输入时辅助翻译和优化
